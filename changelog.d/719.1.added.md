Added a resume-session-modal pane classification so the mailman no longer pastes deliveries into Claude Code's session-resume choice modal (`#719`). When `claude` is launched with more than one resumable session and no `--resume <id>` (a raw launch that bypasses `chamber-claude.sh`'s UUID picker), Claude Code shows a full-screen search modal; a live pane sitting at it looked deliverable (`pane live`, coarsely idle), so paste-and-Enter was consumed as the modal's search input and Enter selected a session — the multi-hour queue-silence the tracker anchors. The new `capturedResumeModal` predicate classifies such a pane `StateAwaitingOperator` (paste-unsafe), so the observe-gate defers it and the pre-paste safety belt aborts. Detection is structural, not a bare substring: the modal's footer legend (`Type to Search · Enter to select · Esc to clear`) is prose-collidable — the tracker dispatch itself quotes it, and a whole-pane match would defer ALL delivery to any pane merely displaying such a message (the `#647` outage class, re-measured for the neighbouring `AwaitingOperatorMarker` in `#852`). The match instead requires TWO companions a prose quote cannot supply: the box-drawing search widget (a `│` box-border with the `⌕` search glyph on one row — MEASURED 0 of 19355 live bus messages carry it, including the 2 that quote the header, footer AND glyph), AND live-scope (no composer `❯` prompt below the footer, so a faithful full-modal capture quoted above a live prompt is still rejected — the durable guard as the marker inevitably enters discussion). The check is placed BEFORE the frame-change classifier branch, mirroring the compaction check and for the same reason: the modal's relative-time entries (`17 seconds ago`) tick across the 200 ms capture window, and the frame-change branch would otherwise read the tick as `StateWorking`, which is paste-SAFE — delivering into the modal. Claude-only (codex parks it via an empty `ResumeModalMarker`); verified against a byte-captured Pilot fixture plus a negative-control corpus of real bus messages that quote the markers, with each structural gate mutation-pinned.
